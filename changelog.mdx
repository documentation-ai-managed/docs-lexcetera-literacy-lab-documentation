## v1.2.0 - 2024-10-15

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for group literacy sessions, allowing up to 10 users to co-edit career skill workbooks.
- Introduced multiliteracy dashboard with interactive modules for digital, visual, and information literacies.
- New API endpoint for integrating Literacy Lab progress into external LMS systems.

## Improvements

- Optimized lesson loading times by 40%, ensuring smoother mobile experiences.
- Enhanced career pathway recommendations using updated labor market data from 2024 sources.

## Bug Fixes

- Fixed sync issues in offline mode for workbook progress tracking.
- Resolved accessibility conflicts with screen readers in multiliteracy exercises.

</Update>
