## Overview

Lexcetera Literacy Lab builds on multiliteracy to empower adult learners with career-ready skills. You engage with dynamic content that blends digital, critical, and communicative literacies. This approach supports personalized growth in professional environments.

<Callout kind="info">
Multiliteracy expands beyond reading and writing to include multimodal communication across digital and real-world contexts.
</Callout>

## Defining Multiliteracy

Traditional literacy focuses on text, but multiliteracy integrates multiple modes like visuals, audio, and interactivity. You develop skills to navigate complex information landscapes.

## Personalized Learning Pathways

You create pathways tailored to your career goals. The system assesses your baseline and recommends modules.

<Steps>
  <Step title="Assess Skills" icon="clipboard-list">
    Complete the initial multiliteracy quiz.
  </Step>
  <Step title="Select Goals" icon="target">
    Choose career focus like tech or management.
  </Step>
  <Step title="Build Pathway" icon="git-branch">
    System generates sequence of modules.
  </Step>
  <Step title="Track Progress" icon="trending-up">
    Monitor mastery with analytics dashboard.
  </Step>
</Steps>

<CodeGroup tabs="JSON,TypeScript">
```json
{
  "learnerId": "user-123",
  "goals": ["digital-literacy", "leadership"],
  "modules": [
    {
      "id": "mod-101",
      "type": "video-interactive",
      "duration": "45m",
      "prerequisite": "quiz-score>80"
    }
  ],
  "progress": 65
}
```
```typescript
interface LearningPathway {
  learnerId: string;
  goals: string[];
  modules: Module[];
  progress: number;
}

const pathway: LearningPathway = {
  learnerId: "user-123",
  goals: ["digital-literacy", "leadership"],
  modules: [{ id: "mod-101", type: "video-interactive" }],
  progress: 65
};
```
</CodeGroup>
